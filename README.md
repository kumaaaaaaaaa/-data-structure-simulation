# 자료구조 시뮬레이터
## 소개
자료구조 알고리즘을 확인할 수 있는 시뮬레이터.

## 목적
자료구조 알고리즘의 이해를 위해 다양한 알고리즘을 하나의 시뮬레이터로 구현.

## 개발과정
각 주차마다 시뮬레이터를 구현하여 완성된 시뮬레이터들을 하나로 묶어 자료구조 시뮬레이터를 제작하였음.
